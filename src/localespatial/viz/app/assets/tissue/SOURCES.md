# Tissue zoom — image sources

Every layer of the organ-to-cell dive is a **real** open-licensed image (nothing is
generated). Sourced from Wikimedia Commons, resized/cropped for the web.

The four histology plates are the same disease (ductal carcinoma) at increasing
magnification, so the dive is a genuine, continuous macro-to-micro path through real
breast-cancer tissue. CC BY 3.0 requires attribution (shown on screen); CC0 does not
but is credited anyway.
