The viewer renders a real, externally-sourced CC-BY cell model. Nothing here is generated.

Get a free Sketchfab API token, then:

export SKETCHFAB_TOKEN=...
python scripts/fetch_assets.py

That writes assets/cell.glb and assets/attribution.json. See assets/ASSETS.md for the full verified sourcing list, or pass ?model=<url> to load any .glb/.gltf/.obj/.stl directly.
